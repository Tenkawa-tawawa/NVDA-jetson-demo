# NVDA-jetson-demo

每一節規劃**簡報說明 (15~20分鐘)+程式實作 (40~45分鐘)**

1. **說明簡報：**：由講師提供簡報草稿/參考資源，再由助教協助整理美化簡報 
2. **程式實作：**：由本專案務提供Demo Scripts，提供學生實作練習的模板

## **教學單元**
**一：Jetson Orin 硬體架構與運算核心解析**

第一節、Arm Cortex CPU 架構與運算特性
第二節、NVIDIA GPU 演進與 CUDA 加速原理
第三節、NVIDIA DLA 深度學習加速器應用

## **教學單元二：深度學習框架與大型語言模型部署**

第四節、PyTorch 基本操作與模型推論流程
第五節、Hugging Face 平台與模型下載/轉換
第六節、TensorRT-LLM Engine 編譯與量化選項

## **教學單元三：Jetson Orin GPIO 與外部介面開發**

第七節、GPIO 控制與程式設計
第八節、I/O 介面（I2C/SPI/UART）基礎應用
第九節：Ultralytics YOLO 手勢辨識與猜拳遊戲 案例
